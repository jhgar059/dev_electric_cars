Propósito del Proyecto

El proyecto "Electric Cars Database" busca ser una plataforma integral y fácil de usar que centralice información sobre vehículos eléctricos y su infraestructura de carga.

  • Información Detallada de Autos: Proporcionar datos claros y actualizados sobre marcas, modelos, capacidades de batería, autonomía y otros detalles relevantes de autos eléctricos.
  • Infraestructura de Carga: Ofrecer información exhaustiva sobre estaciones de carga, incluyendo ubicación, tipos de conectores, potencia, horarios y costes.
  • Dificultad de Carga: Evaluar y registrar la facilidad o complejidad de la carga para distintos modelos, incluyendo tiempo de carga y requisitos de instalación.
  • Visualización de Datos Clave: Ofrecer una sección dedicada a estadísticas y gráficos interactivos que presenten tendencias del mercado, distribuciones y otros datos relevantes de manera visual.
  • Historial de Registros Eliminados: Mantener un registro de elementos eliminados para auditoría y referencia, garantizando la integridad de los datos.

Este proyecto es una demostración práctica de desarrollo de software moderno, utilizando FastAPI para la API backend, SQLAlchemy para la gestión de la base de datos y Jinja2 junto con Bootstrap para el frontend. Es una herramienta valiosa para cualquiera interesado en el emocionante mundo de la movilidad eléctrica.

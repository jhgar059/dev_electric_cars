Acerca del Proyecto "Electric Cars Database"

El proyecto "Electric Cars Database" es una iniciativa para crear una plataforma web integral dedicada a la recopilación y gestión de información sobre vehículos eléctricos (EVs) y su infraestructura de carga. Nace de la creciente necesidad de un recurso centralizado que facilite el acceso a datos cruciales para entusiastas, compradores potenciales y profesionales de la industria.

Nuestro objetivo es simplificar la búsqueda y comprensión de las especificaciones de los vehículos, detalles de las estaciones de carga y el análisis de la experiencia de carga. Implementamos un sistema robusto que permite:

  • Gestión de Autos Eléctricos: Registrar, consultar, actualizar y eliminar información detallada de diversos modelos de EVs, incluyendo marca, modelo, año, capacidad de batería y autonomía.
  • Administración de Estaciones de Carga: Mantener un inventario de estaciones, con datos como ubicación, tipo de conector, potencia y disponibilidad.
  • Análisis de Dificultad de Carga: Evaluar y registrar la facilidad o complejidad de la carga para distintos modelos, incluyendo tiempo de carga y requisitos de instalación.
  • Visualización de Datos Clave: Ofrecer una sección dedicada a estadísticas y gráficos interactivos que presenten tendencias del mercado, distribuciones y otros datos relevantes de manera visual.
  • Historial de Registros Eliminados: Mantener un registro de elementos eliminados para auditoría y referencia, garantizando la integridad de los datos.

Este proyecto es una demostración práctica de desarrollo de software moderno, utilizando FastAPI para la API backend, SQLAlchemy para la gestión de la base de datos y Jinja2 junto con Bootstrap para el frontend. Es una herramienta valiosa para cualquiera interesado en el emocionante mundo de la movilidad eléctrica.

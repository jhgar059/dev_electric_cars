Nuestro Objetivo

El proyecto "Electric Cars Database" busca ser una plataforma integral para la gestión y consulta de datos relacionados con vehículos eléctricos y su infraestructura de carga. Nuestras metas principales incluyen:

  • Gestión Completa de Autos: Permitir la administración (CRUD) de vehículos eléctricos, incluyendo detalles como marca, modelo, año, capacidad de batería, autonomía y disponibilidad.
  • Gestión de Estaciones de Carga: Facilitar el registro y consulta de estaciones, abarcando ubicación, tipo de conectores, potencia, acceso público y horarios.
  • Evaluación de Dificultad de Carga: Evaluar y registrar la facilidad o complejidad de la carga para distintos modelos, incluyendo tiempo de carga y requisitos de instalación.
  • Visualización de Datos Clave: Ofrecer una sección dedicada a estadísticas y gráficos interactivos que presenten tendencias del mercado, distribuciones y otros datos relevantes de manera visual.
  • Historial de Registros Eliminados: Mantener un registro de elementos eliminados para auditoría y referencia, garantizando la integridad de los datos.

Este proyecto es una demostración práctica de desarrollo de software moderno, utilizando FastAPI para la API backend, SQLAlchemy para la gestión de la base de datos y Jinja2 junto con Bootstrap para el frontend. Es una herramienta valiosa para cualquiera interesado en el emocionante mundo de la movilidad eléctrica.
